### 개발 동기

디스코드와 카카오톡 같은 채팅 애플리케이션을 직접 만들어 보고 싶어서 개발하게 되었다.

### 기대효과

-   실시간 커뮤니케이션: `Socket.io`를 사용해서 각 사용자별 실시간으로 데이터를 주고 받을 수 있다.
-   채팅에서 더 나아가 화면을 캡쳐해 전송하거나, `WebRTC` 등을 사용하면 원격 작업까지 가능해진다.
